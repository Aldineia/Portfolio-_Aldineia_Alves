Sobre Mim

Engenheira de Software em Formação e Modelagem de Sistemas

Sou estudante de Engenharia de Software na Universidade Anhanguera-Uniderp. Minha base técnica concentra-se em todo o ciclo de vida do desenvolvimento, com ênfase na Análise de Requisitos, Modelagem UML e Análise Orientada a Objetos. Tenho experiência na implementação de soluções robustas em Java e Python e foco no desenvolvimento de sistemas com alta escalabilidade.

Práticas realizadas em MongoDB (criação e manipulação de bancos orientados a documentos) e CloudSim (simulação de alocação de recursos em nuvem) reforçaram minha capacidade de gerenciar dados complexos e otimizar a arquitetura de sistemas. Meu trabalho inclui o uso de ferramentas como Visual Paradigm para modelagem e MySQL Workbench para programação SQL.

Meus Projetos

Trabalhos práticos detalhados que demonstram minhas competências em Engenharia de Software.

Modelagem UML: Sistema Bancário

Desenvolvimento da análise de requisitos e modelagem de um Diagrama de Caso de Uso completo para um sistema bancário, utilizando o Visual Paradigm. Focado na identificação de atores e funcionalidades essenciais (saque, depósito, abertura/encerramento de conta).

UML Análise de Requisitos Visual Paradigm Modelagem de Sistemas

Modelagem e Manipulação NoSQL (MongoDB)

Criação e manipulação de um banco de dados orientado a documentos (`lojadb`) utilizando MongoDB e o cliente Compass. O projeto evidenciou a flexibilidade e escalabilidade do modelo NoSQL para dados heterogêneos.

MongoDB NoSQL Banco de Documentos MongoDB Compass

Programação e Desenvolvimento SQL

Criação e manipulação completa de um banco de dados relacional (tabelas, inserção de dados, consultas e Visões) utilizando MySQL Workbench e comandos SQL eficientes, garantindo a integridade dos dados.

SQL MySQL Workbench Consultas Avançadas Programação de BD

Fundamentos de IA: Rede Neural em Python

Implementação de uma rede neural de uma camada em Python, utilizando a função sigmoide, para a tarefa de classificação binária (lógica AND). Foco no treinamento supervisionado e ajuste de pesos para aprendizado de padrões.

Python Inteligência Artificial Rede Neural Classificação Binária

Simulação de Gerenciamento em Nuvem (CloudSim)

Instalação e execução de um ambiente de simulação em Java (NetBeans) utilizando a plataforma CloudSim. Demonstra a capacidade de análise de desempenho e tomada de decisão na alocação de recursos em nuvem.

CloudSim Computação em Nuvem Java Simulação

Modelagem de Sistemas (DER e Diagramas de Classe)

Aplicação de Análise Orientada a Objetos (AOO) para criar um Diagrama de Classes (Locação de Automóveis) e de Modelagem de Dados para desenvolver um DER (Sistema de Empréstimo de Livros), utilizando Visual Paradigm e MySQL Workbench.
